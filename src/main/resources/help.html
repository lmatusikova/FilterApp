Filter application

How to work with application

  1. Load images from the given directory

    2. Only images with following suffixes are allowed:

    • jpg, JPG, png, PNG
  2. Choose filter:
    • Grayscale filter
    • Threshold filter
    • Edge detection filter
  3. You can pre-process selected image or all images from the given directory

  4. Then you can save pre-process image/images

About filters

  1. Grayscale filter

    2. Converts image to the gray-scale tones.


  2. Threshold filter

    3. You can choose from these methods of computing threshold function:

    • Otsu method
    • Entropy
  3. Edge detection filter

    4. You can detect edges with following operators:

    • Roberts operator
    • Sobel operator
    • Prewitt operator
    • Laplace operator
